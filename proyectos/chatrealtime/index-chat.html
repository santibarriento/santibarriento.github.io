Chat en Tiempo Real

Una aplicación web moderna y funcional para mensajería instantánea desarrollada con Spring Boot y WebSockets.

Descripción General

Esta aplicación permite a los usuarios comunicarse en tiempo real mediante una interfaz limpia e intuitiva. Los mensajes se sincronizan instantáneamente entre todos los participantes conectados, ofreciendo una experiencia fluida y dinámica.

La arquitectura se basa en WebSockets para la comunicación bidireccional, utilizando SockJS como fallback y STOMP.js para el manejo eficiente de mensajes y suscripciones.

Funcionamiento Técnico

Conexión WebSocket

Establece un canal persistente y de baja latencia entre el cliente y el servidor al iniciar la aplicación.

Protocolo STOMP

Gestiona el enrutamiento de mensajes hacia destinos específicos (topics) para una distribución ordenada.

Suscripción Pub/Sub

Los clientes se suscriben a /topic/messages para recibir actualizaciones automáticas sin recargar.

Envío Asíncrono

Los mensajes se envían como objetos JSON y se procesan en el backend para su difusión inmediata.

Características Principales

  • Envío y recepción instantánea
  • Identificación de usuarios
  • Historial de chat con scroll automático
  • Interfaz responsiva y moderna

Stack Tecnológico
